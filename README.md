Mac电脑如果安装了 [home-brew](https://brew.sh/)，可直接输入

```
brew install tree
```

安装完成后，直接用命令即可显示项目目录结构

```
tree project (your project)
```

for example 

```
tree Daily
```



更多命令的使用可以查看tree --help
